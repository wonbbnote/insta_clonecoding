## 2. 개발해야 하는 기능 및 역할 분담
- 김성호 [로그인페이지] 회원가입/로그인/로그아웃
- 김주훈 [메인페이지] 사진 업로드 및 좋아요/댓글 기능
- 박진우 [마이페이지] 게시물 UI 및 게시물 활동
- 최희원 [팔로우/팔로잉] 팔로잉 및 팔로우 수 및 구독 정보 저장
